 OBJETOS GLOBAIS:

    São objetos padrão da linguagem javascript que ficam sempre disponíveis para utilizarmos.


    Atenção: não confundir com o objeto de escopo global window do navegador (ou escopo global de outros ambientes)


    Alguns Exemplos:

        Object:
             let pessoa = Object()
             pessoa -> {}

        arrays
            let lista = Array()
            lista -> []

        String
            let texto = String()
            texto = ''

        Number
            let numero = Number()
            numero -> 0

        Date
            let data = Date()
            data -> Exemplo: 09/06/2025 (sendo essa a data atual)

        Math



    Podem ser usados para criar e converter valores.


    Possuem propriedades e funções úteis e frequentemente necessárias.
